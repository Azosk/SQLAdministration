SQL Server 2022 Setup Instructions
1. Install SQL Server 2022 (Developer or Express Edition)
Developer Edition (recommended for development and testing)
Download link: https://aka.ms/sqlserverdl

Features: Full SQL Server engine with all features enabled

Usage: Free for development and testing only, not for production

Express Edition (recommended for lightweight production or learning)
Download link: https://aka.ms/expressdl

Features: Free, limited database size (10 GB max), limited RAM and CPU

Usage: Production use allowed for small apps

Steps:

Download the installer from the links above.

Run the installer and follow the setup wizard.

Choose the edition (Developer or Express).

Set a strong password for the sa (system administrator) account.

Complete installation.
